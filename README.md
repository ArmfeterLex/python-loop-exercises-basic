# Python Loop Exercises Basic

Коллекция базовых упражнений на Python для отработки циклов for, range и условных операторов.

## Задачи

### 1️⃣ Вывод чисел от A до B (возрастание)
Пользователь вводит A и B. Если A ≤ B, программа выводит все числа от A до B включительно.

### 2️⃣ Вывод чисел от A до B (оба направления)
Если A < B — вывод по возрастанию, если A > B — по убыванию, если A = B — выводит одно число.

### 3️⃣ Нечетные числа от A до B (убывание)
Пользователь вводит A и B (A > B). Программа выводит все нечетные числа от A до B включительно в порядке убывания.

### 4️⃣ Сумма 10 чисел
Программа запрашивает 10 чисел и выводит их сумму.

### 5️⃣ Сумма N чисел
Пользователь вводит количество чисел N, затем сами числа. Программа выводит их сумму.

### 6️⃣ Сумма кубов от 1 до n
Вычисление суммы кубов: 1³ + 2³ + 3³ + ... + n³

### 7️⃣ Факториал числа n
Вычисление n! = 1 × 2 × 3 × ... × n (с обработкой случая n = 0)

### 8️⃣ Сумма факториалов от 1! до n!
Вычисление 1! + 2! + 3! + ... + n!

### 9️⃣ Подсчет нулей
Пользователь вводит N чисел. Программа считает, сколько среди них нулей.

## Требования

- Python 3.x

## Примечание

Все задачи построены по принципу "от простого к сложному". Начинаются с простого вывода чисел, затем добавляются условия, потом математические вычисления. Отлично подходят для первого знакомства с циклами.
